
# Litecom2MQTT Add-on Repository

Detta är ett **Home Assistant add-on repository** innehållande add-on:et `Litecom2MQTT`.

## Struktur
```
/ (repo root)
├─ repository.json
└─ litecom2mqtt/
   ├─ config.json
   └─ README.md
```

## Använd som URL i Home Assistant
Lägg till detta repo i *Settings → Add-ons → Add-on Store → ⋮ → Repositories* med din GitHub‑URL.
